🧠 Day 19 Notes – Graph Shortest Path + CNN Introduction:

📌 SLOT 01: Graph Shortest Path (Dijkstra + BFS Recap):

1️⃣ Graph Recap
Graph ek data structure hai jisme nodes (vertices) aur edges hote hain.
👉 Example:
Social network, maps, internet connections

2️⃣ BFS Recap (Breadth First Search)
BFS level-by-level traversal karta hai.
👉 Queue use hoti hai
👉 Unweighted graph me shortest path nikal sakta hai
👉 Simple idea:
Pehle neighbors visit karo

3️⃣ What is Dijkstra Algorithm?
Dijkstra algorithm weighted graph me shortest path find karta hai.
👉 Condition → weights positive hone chahiye
👉 Real-life example → Google Maps route

4️⃣ How Dijkstra Works (Idea)
Start node choose karo
Distance initialize karo
Minimum distance node pick karo
Neighbors update karo
👉 Repeat until all nodes visited

5️⃣ BFS vs Dijkstra Difference
BFS → unweighted graph
Dijkstra → weighted graph
👉 Memory trick:
Weight hai → Dijkstra
Weight nahi → BFS

📌 SLOT 02: CNN (Convolutional Neural Network) Introduction:

1️⃣ What is CNN?
CNN ek deep learning model hai jo specially images ke liye use hota hai.
👉 Image patterns detect karta hai

2️⃣ Why CNN is Used?
Image classification
Face detection
Medical imaging
Object detection
👉 CNN = computer vision king

3️⃣ Convolution Layer Idea
Convolution layer filters use karke image features detect karti hai.
👉 Example features:
Edges, shapes, textures
👉 Simple:
Image par scanner chalana

4️⃣ Pooling Layer Idea
Pooling layer image size reduce karti hai.
👉 Max pooling → important value pick
👉 Benefit → computation kam
👉 Simple:
Image compress karna

5️⃣ CNN Basic Flow
Image → Convolution → Pooling → Fully connected → Output
👉 Step-by-step feature extraction

🔁 Quick Revision
👉Graph → nodes + edges
👉BFS → unweighted shortest path
👉Dijkstra → weighted shortest path
👉CNN → image learning model
👉Convolution → feature detection
👉Pooling → size reduction
