🧠 Day 20 Notes – Graph Algorithms + CNN Training:

📌 SLOT 01: Graph Algorithms (DFS, Cycle Detection, Topological Sort):

1️⃣ Depth First Search (DFS)
DFS ek graph traversal technique hai jisme hum ek path ko depth tak explore karte hain aur phir backtrack karte hain.
👉 Recursion ya stack use hota hai
👉 Deep exploration strategy

2️⃣ DFS vs BFS
DFS → depth-first exploration
BFS → level-wise exploration
👉 Memory tip:
DFS = Deep
BFS = Breadth

3️⃣ Cycle Detection in Graph
Cycle ka matlab graph me loop exist karta hai.
👉 Directed graph → DFS + recursion stack
👉 Undirected graph → DFS + parent check
👉 Cycle detection network reliability me useful hota hai

4️⃣ Topological Sort
Topological sort directed acyclic graph (DAG) ka linear ordering hota hai.
👉 Condition → graph acyclic hona chahiye
👉 Use → task scheduling, dependency resolution
👉 Idea → node tab print hota hai jab uske dependencies complete ho jaye

📌 SLOT 02: CNN Training & Evaluation

1️⃣ Dataset Splitting
Model training se pehle dataset ko split karte hain:
Train set → learning
Test set → evaluation
👉 Common split → 80:20

2️⃣ Loss Function
Loss function prediction error measure karta hai.
👉 Loss kam → model better
👉 CNN me cross-entropy commonly use hota hai

3️⃣ CNN Training Steps
Forward pass → prediction
Loss calculation → error
Backpropagation → gradient calculation
Weight update → learning
👉 Ye training loop hota hai

4️⃣ Overfitting in CNN
Overfitting tab hota hai jab model training data me perfect but new data me poor perform kare.
👉 Solution:
More data
Regularization
Dropout

5️⃣ Model Evaluation
Model ko evaluate karte hain:
Accuracy
Loss
Validation performance
👉 Goal → generalization

🔁 Quick Revision
👉DFS → depth traversal
👉Cycle → graph loop
👉Topological sort → DAG ordering
👉CNN training → forward + backward pass
👉Overfitting → poor generalization
